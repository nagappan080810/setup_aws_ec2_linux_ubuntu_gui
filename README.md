## Setup Remote Desktop on Linux
### Run ./rdp_gui.sh or the following commands in Ubuntu.
- sudo apt update -y 
- sudo apt install lxde -y
- sudo start lxdm
- sudo apt install xrdp -y
- sudo passwd ubuntu

- (optional) sudo apt install firefox
